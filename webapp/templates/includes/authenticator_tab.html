RingCentral PKCE Connection Guide (Layer 2)

Securely authenticate your RingCentral app to interact with the platform APIs.

Instructions for PKCE Authorization:

  • CRITICAL STEP: OAuth Redirect URI

    In the RingCentral Developer Console, set the OAuth Redirect URI to this exact URL:

    {{ RC_REDIRECT_URI }}
  • Paste the Client ID from your RingCentral application into the input field below.
  • Click Connect. This will redirect you to RingCentral's login page to grant access to the customer account.
  • After successful login on RingCentral, you will be redirected back, and the status above will change to Connected.

API Endpoint: N/A (Client-side flow handled by /api/rc/initiate-auth)

Rate Limit: Low (only used for initiating the connection).

Connection Form

Enter the customer's application Client ID to initiate authorization.
