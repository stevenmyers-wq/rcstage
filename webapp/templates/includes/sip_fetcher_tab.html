SIP Credentials Fetcher

Retrieve the SIP provision information for a device for manual configuration.

Instructions & Notes:

  • Prerequisite: RC Account must be connected via Impersonation or the Main Bridge.
  • Function: This retrieves the SIP provision information for a device, typically used for manual softphone or physical phone configuration.
  • API Endpoint: GET /restapi/v1.0/account/~/extension/<extId>/sip-provision
  • Rate Limit: Moderate. (Typically 40-50 calls per minute, per API Key/App)
  • Execution Time: Fast (sub-second).
