SIP Credentials Fetcher

Retrieve the SIP provision parameters required for third-party endpoints or softphone endpoints manually.

Instructions & Session Parameters:

  • Prerequisite: Context token must be assigned via active tenant bridging configurations.
  • Method Profile: Submits an encapsulated gateway request to GET /restapi/v1.0/account/~/extension/<extId>/sip-provision.
  • Security Notice: SIP configuration parameters are highly sensitive. Credentials extracted are contained to active runtime browser contexts and are never mirrored down onto permanent server database profiles.
  • Rate Limit: Moderate throttling applies. (40 requests / minute max baseline).
