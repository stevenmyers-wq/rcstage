Bulk Hours & Routing Management

Mass configuration changes for Sites and Queues business hours, standard routing (In/Out of Hours), and custom rules.

Part 1: Business Hours (In-App)

  1. Fetch Site or Queue Hours.
  2. Select rows using checkboxes.
  3. Use the Bulk Edit panel below to change schedules.
  4. Click Apply Changes to push to RingCentral.

Part 2: Routing Rules (CSV Method)

  1. Fetch Default (BH/AH) or Custom Rules.
  2. Download as CSV. Type the new Extension Number in the ActionTarget column.
  3. Upload the modified file to apply changes in bulk.
