Account Migration Tool

Export all programming data, user settings, announcements, queues, IVR maps, and all audio files from the currently connected account into a ZIP archive, and restore them to a new tenant.

Warning: You can only be connected to one account at a time. Ensure you are connected to the Source Account when exporting, and the Destination Account when importing.

1. Export Source Account

Extract deep configurations and audio settings.

Scans the tenant to generate a ZIP containing machine-readable JSON, Excel audit, Cost Centers, Routing Rules, User Business Hours, Announce/Message settings, and all Custom Audio files.

2. Import to Destination

Restore configurations from a ZIP archive.

Upload the ZIP file. The system will map Cost Centers, create Sites, IVRs, Queues, Announcements, Message-Only extensions, Park Locations, Paging Groups, and map audio files.
